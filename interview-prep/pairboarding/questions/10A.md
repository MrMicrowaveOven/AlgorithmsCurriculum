**Write a function that takes an integer and returns it in binary form.**

  ```ruby
    def binary(num)
      result = []
      until num == 0
        result.unshift num % 2
        num /= 2
      end
      result.join
    end
  ```
 
**Implement factorial with and without recursion.**

  ```ruby
      def rec_fac(num)
        return 1 if num == 1
        num * rec_fac(num - 1)
      end
    
      def iter_fac(num)
        i = num - 1
        while i > 0
          num *= i
          i -= 1
        end
        num
      end
  ```
